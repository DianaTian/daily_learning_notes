## sort
内置列表方法 list.sort() 原地修改列表，而内置函数 sorted() 由可迭代对象**新建**有序列表。

排序的基础知识
普通的升序排序非常容易：只需调用 sorted() 函数。它返回新有序列表：

```python
>>> sorted([5, 2, 3, 1, 4])
[1, 2, 3, 4, 5]
```

亦可用 list.sort() 方法。它原地修改原列表（并返回 None 以避免混淆）。
```python
>>> a = [5, 2, 3, 1, 4]
>>> a.sort()
>>> a
[1, 2, 3, 4, 5]
```
另一个区别是 list.sort() 方法只为列表定义，而 sorted() 函数接受任何可迭代对象。

```
>>> sorted({1: 'D', 2: 'B', 3: 'B', 4: 'E', 5: 'A'})
[1, 2, 3, 4, 5]
```
